# 12306车次列表页面分析

## 页面概述
车次列表页面是用户在首页搜索车票后跳转的结果页面，URL为：`https://kyfw.12306.cn/otn/leftTicket/init?linktypeid=dc`

## 页面结构分析

### 1. 页面头部
- **导航栏**：包含12306 logo、搜索框、用户登录状态
- **主导航菜单**：首页、车票、团购服务、会员服务、站车服务、商旅服务、出行指南、信息查询

### 2. 查询条件展示区
- **出发地和目的地**：显示当前查询的起始站和终点站，支持交换
- **出发日期**：显示选中的出发日期
- **乘客类型**：成人/学生选择
- **车次类型**：高铁/动车选择

### 3. 日期选择区
- **日期导航**：显示连续15天的日期选项
- **当前日期高亮**：选中日期用蓝色背景标识
- **日期格式**：MM-DD 格式，显示星期几

### 4. 筛选条件区
- **发车时间筛选**：
  - 00:00-06:00（深夜）
  - 06:00-12:00（上午）  
  - 12:00-18:00（下午）
  - 18:00-24:00（晚上）

- **车次类型筛选**：
  - 全部
  - GC-高铁/城际
  - D-动车
  - Z-直达
  - T-特快
  - K-快速
  - 其他
  - 复兴号智能动车组

- **出发车站筛选**：全部 + 具体车站列表
- **到达车站筛选**：全部 + 具体车站列表
- **车次席别筛选**：全部 + 各种座位类型

### 5. 显示选项
- **显示折扣车次**：复选框
- **显示积分兑换车次**：复选框  
- **显示全部可预订车次**：复选框

### 6. 车次列表
#### 表头
- 车次、出发站、到达站、出发时间、到达时间、历时
- 各种座位类型：商务座、特等座、优选一等座、一等座、二等座、二等包座、高级软卧、软卧/动卧、一等卧、硬卧、二等卧、软座、硬座、无座、其他
- 备注

#### 车次信息行
- **车次号**：如G1407、G2788等，带有车次类型标识
- **出发/到达站**：显示具体站名
- **时间信息**：出发时间、到达时间、运行时长
- **座位信息**：
  - 有票：显示票数或"有"
  - 无票：显示"--"
  - 候补：显示"候补"
- **预订按钮**：蓝色"预订"按钮

### 7. 页面底部
- **友情链接**
- **版权信息**
- **官方微信/微博二维码**

## UI设计特点

### 颜色方案
- **主色调**：蓝色系（#0066CC）
- **辅助色**：灰色系、绿色（有票）、橙色（候补）
- **背景色**：白色为主，浅灰色分割

### 字体规范
- **主要字体**：微软雅黑、宋体
- **字号层级**：
  - 标题：16px-18px
  - 正文：14px
  - 辅助信息：12px

### 布局特点
- **响应式设计**：支持不同屏幕尺寸
- **表格布局**：车次列表采用表格形式
- **固定头部**：导航和筛选条件固定在顶部

### 交互设计
- **筛选联动**：多个筛选条件可同时生效
- **实时更新**：筛选条件变化时列表实时更新
- **排序功能**：支持按时间、价格等排序
- **分页加载**：大量数据分页显示
