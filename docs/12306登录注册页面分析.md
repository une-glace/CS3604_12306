# 12306登录和注册页面分析

## 登录页面分析

### 页面结构
基于12306官方登录页面 `https://kyfw.12306.cn/otn/resources/login.html` 的分析：

#### 1. 页面布局
- **顶部导航栏**：包含12306 logo和主要导航链接
- **主体内容区**：居中的登录表单
- **底部信息**：版权信息和相关链接

#### 2. 登录表单组件
- **用户名输入框**
  - 支持用户名或邮箱登录
  - 验证规则：6-30位字母、数字或"_"，字母开头
  - 邮箱格式验证
- **密码输入框**
  - 最小长度6位
  - 密码强度提示
- **验证码区域**
  - 图形验证码（点击式验证码）
  - 刷新验证码功能
- **登录选项**
  - "记住用户名"复选框
  - "自动登录"复选框
- **登录按钮**
  - 主要操作按钮，蓝色背景
- **其他功能链接**
  - "忘记用户名"链接
  - "忘记密码"链接
  - "立即注册"链接

#### 3. 验证码机制
- 采用点击式图形验证码
- 需要用户点击图片中的指定文字
- 验证码校验接口：`/passport/captcha/captcha-check`
- 验证成功后才能进行登录验证

#### 4. 登录流程
1. 用户输入用户名/邮箱和密码
2. 完成验证码验证
3. 提交登录请求到 `/passport/web/login`
4. 验证成功后跳转到主页或指定页面

### UI设计特点
- **色彩方案**：以蓝色为主色调，白色背景
- **表单设计**：简洁的输入框设计，带有明确的标签
- **错误提示**：红色文字提示，配有错误图标
- **响应式设计**：适配不同屏幕尺寸

## 注册页面分析

### 页面结构
基于12306官方注册页面 `https://kyfw.12306.cn/otn/regist/init` 的分析：

#### 1. 页面布局
- **顶部导航**：与登录页面保持一致
- **步骤指示器**：显示注册流程的当前步骤
- **表单区域**：分步骤的注册表单
- **底部信息**：服务条款和隐私政策链接

#### 2. 注册表单字段

##### 账户信息部分
- **用户名**
  - 必填项，6-30位字母、数字或"_"
  - 字母开头，实时验证唯一性
- **登录密码**
  - 必填项，最少6位字符
  - 密码强度指示器
- **确认密码**
  - 必填项，与登录密码一致性验证

##### 个人信息部分
- **证件类型**
  - 下拉选择：中国居民身份证、外国人永久身份证、港澳台居民身份证
- **姓名**
  - 必填项，证件上的中文姓名
  - 用于身份证核验
- **证件号码**
  - 必填项，身份证号码验证
  - 实时格式验证和校验码验证
- **邮箱**
  - 必填项，邮箱格式验证
  - 用于找回密码和接收通知
- **手机号码**
  - 必填项，11位手机号格式验证
  - 短信验证码验证
- **旅客类型**
  - 选择：成人/儿童

#### 3. 验证机制
- **实时验证**：输入框失焦时进行格式验证
- **身份证验证**：使用算法验证身份证号码的有效性
- **手机验证码**：发送短信验证码进行手机号验证
- **邮箱验证**：邮箱格式的正则表达式验证

#### 4. 注册流程
1. 填写账户基本信息
2. 填写个人身份信息
3. 手机号验证码验证
4. 同意服务条款
5. 提交注册申请
6. 注册成功，跳转到登录页面
